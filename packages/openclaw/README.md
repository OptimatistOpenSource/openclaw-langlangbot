# @optimatist/langlangbot-openclaw

OpenClaw channel plugin for LangLangBot — Operator chat, streaming replies, and exec/plugin
approvals.

Connects OpenClaw to a local LangLangBot sidecar at `https://127.0.0.1:9528` (default).

## Install

```bash
openclaw plugins install @optimatist/langlangbot-openclaw@latest
openclaw gateway restart
```

Full setup (sidecar + plugin + pairing): see [docs/INSTALL.md](../../docs/INSTALL.md).

Requires Node.js ≥20 and OpenClaw ≥2026.5.7.
